# Introduction

iTVEncoder is a real time encoder using gstreamer.

# Install from source

## Prerequisites

gnome-common

autoconf

automake

libtool

gstreamer-devel

gstreamer-plugins-base-devel

## Compile and install

    ./autogen.sh
    ./configure (--help)
    make
    make install
